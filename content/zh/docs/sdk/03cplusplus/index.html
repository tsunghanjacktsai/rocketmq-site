C++ 客户端

版本说明​

本章节介绍使用 Apache RocketMQ 5.0 gRPC 协议 C++ SDK 收发消息。

信息
  • 本示例代码基于gRPC 协议 SDK 构建，因此需要服务端至少升级到5.0版本，并启用 gRPC Proxy 才可兼容。关于Proxy的部署参考快速指南。
  • 如果您使用的是 Remoting 协议 SDK，建议参考之前4.x 版本的示例代码即可运行.如何识别您使用的 SDK 类型，参考概览。
